
vary.vim
========

A script to remove spaces in end of line

####Install

```
	$ cd ~/.vim/bundle
	$ git clone https://github.com/wbluecat/vary.vim
```

####Usage

You need to set the following two types of variables to ~/.vimrc:


```
	set tabstop=4
	map <F11> :TVaryVim<CR>
	#into spaces <tab> type of language
```
